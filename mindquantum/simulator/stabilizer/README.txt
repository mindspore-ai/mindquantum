requirement list:
    python >= 3.6
    numpy >= 1.21
    mindquantum >= 0.6

features:
    (i)  CHP gates apply on the tableau
    (ii) Clifford decomposition: Aaronson-Gottesman method
    (iii) Clifford multiplication
    (iv) 2-qubits Clifford multiplication table using phase separation
    (v) Clifford to MindQuantum circuit, and circuit to Clifford

references:
    S. Aaronson, D. Gottesman, Improved Simulation of Stabilizer Circuits,
        Phys. Rev. A 70, 052328 (2004).  arXiv:quant-ph/0406196
    Sergey Bravyi and Dmitri Maslov. Hadamard-free circuits expose the structure of the Clifford group.  https://arxiv.org/abs/2003.09412v2
