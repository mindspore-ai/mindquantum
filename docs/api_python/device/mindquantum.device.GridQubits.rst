mindquantum.device.GridQubits
=============================

.. py:class:: mindquantum.device.GridQubits(n_row, n_col)

    格点形式的量子比特拓扑结构。

    参数：
        - **n_row** (int) - 格点有多少行。
        - **n_col** (int) - 格点有多少列。

    .. py:method:: n_col()

        获取列数。

        返回：
            int，列数。

    .. py:method:: n_row()

        获取行数。

        返回：
            int，行数。
