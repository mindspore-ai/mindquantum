5. About writing style, please refering to ``tex/Tensorflow_Quantum.pdf``.

## Writing Style

- Inline code

> ``\code{}``

- Link

> ``\fancylink{link}{display name}``

- Equation more than one line

> ``\spliteq{}``

- Euqation reference

>  ``Eq.~\eqref{}``

- Figure reference

> ``Fig.~\ref{}``
