# 黑客松赛题

量子神经网络具有表达能力强、搜索空间大等特点，在特定算法上能够超越经典的神经网络。在本题中，选手探索利用量子经典混合的神经网络来完成手写字体识别任务。

# 代码大致介绍
在’Main‘class里面，’build_encoder‘构建了一种简化版的Amplitude encoder线路；’build_ansatz‘借助’rebuild_circuit‘重建了我们根据Monte Carlo随机采样得到的ansatz线路。
